# phasmo-ev-overlay

Evidences overlay for phasmophobia 

shortcuts : 

Shift + 0 : reset  
Shift + 1 : toggle Emf  
Shift + 2 : toggle Spirit  
Shift + 3 : toggle Writing  
Shift + 4 : toggle Orb  
Shift + 5 : toggle Finger  
Shift + 6 : toggle Temp  
Shift + 7 : toggle Dots  
m : toggle maps  
Shift + Plus : Quit overlay
